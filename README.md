# БК Вилки с Аманом

Лендинг с регистрацией и дашбордом для торговой платформы.

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# или npx
npx serve .
```

Затем перейдите на http://localhost:8000

## Настройка Google Sheets

1. Создайте [новую таблицу](https://sheets.google.com)
2. Меню **Расширения** → **Apps Script**
3. Удалите содержимое `Code.gs` и вставьте код из файла `Code.gs` в этом проекте
4. Сохраните (Ctrl+S)
5. **Развернуть** → **Новое развёртывание** → шестерёнка → **Веб-приложение**
6. Параметры:
   - **Выполнять от имени:** Я
   - **У кого есть доступ:** Все
7. Нажмите **Развернуть** и скопируйте URL
8. Откройте `script.js` и вставьте URL в переменную `GOOGLE_SHEETS_URL`

## Структура

- `index.html` — разметка (лендинг, модалка, дашборд)
- `styles.css` — стили, тёмная тема, анимации, адаптивность
- `script.js` — логика: форма, Google Sheets, переключение страниц
- `Code.gs` — скрипт для Google Apps Script (копировать в Apps Script)
